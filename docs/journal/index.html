for(Bezier &amp;path : *shape.getPaths()) {
  bezierPos = path.getPosition();
  bezierDim = path.getDimensions();
 
  //Move bezier to origin
  path.getCursor()
    -&gt;reset()
    -&gt;translate(-(bezierPos.x + bezierDim.x / 2.0),
                -(bezierPos.y + bezierDim.y / 2.0), 0);
 
  //Move bezier start point to origin
  path.getCursor()
    -&gt;translate(-path.getStartPoint());
 
  //Move besier to lastPos
  path.getCursor()
    -&gt;translate(lastPos);
 
  Bezier pathTemp = path;
  pathTemp.applyCursor();
 
  lastPos = pathTemp.getEndPoint(); }

Le développement du logiciel Images génériques (./i) s’est déroulé pendant 3 semaines de projet tuteuré, regroupant 4 élèves ingénieurs de l’IMAC.

L’équipe est constituée de :
Marie-Lou Barbier, cheffe de projet et développeuse
Valentin Dufois, responsable logiciel  et développeur
Alexane Le Guern, responsable index et développeuse
Kenthyvong Tan, responsable traceur et développeur
